# Dashboard Design Skill

## Purpose

Create local dashboards that make agent progress, productivity, and artifact
quality visible without external services.

## Inputs

- Metrics.
- Artifact paths.
- Status labels.
- Run metadata.

## Outputs

- Local HTML dashboard.
- `dashboard-data.json`.
- Metric cards.
- Review table.

## Procedure

1. Use local inline data only.
2. Show baseline, current value, and status when possible.
3. Keep cards compact and scannable.
4. Include links or paths to generated artifacts.
5. Avoid external scripts, external images, and automatic network calls.
