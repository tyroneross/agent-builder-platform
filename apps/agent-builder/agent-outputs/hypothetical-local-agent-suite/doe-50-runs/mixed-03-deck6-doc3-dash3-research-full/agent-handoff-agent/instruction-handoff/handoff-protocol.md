# Agent Handoff Protocol

## Format

- full

## Share When

- before delegation
- after tool output changes assumptions
- before final artifact assembly
- when uncertainty or permissions change

## Share What

- goal
- inputs
- constraints
- accepted assumptions
- source and claim state
- expected artifacts
- stop conditions
- handoff owner

## Do Not Share

- irrelevant transcript history
- credentials or secrets
- unverified claims without labels
- tool permissions broader than the receiver needs

## Template

- Goal
- Context summary
- Source state
- Claim state
- Inputs
- Tools allowed
- Constraints
- Output contract
- Validation plan
- Stop condition
