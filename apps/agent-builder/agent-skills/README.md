# Agent Skills

Reusable skills for generated agents. Keep them small, auditable, and tied to
measurable behavior.

- `chief-of-staff/100x-productivity-planning.skill.md`

- `shared/claim-verification.skill.md`

- `shared/dashboard-design.skill.md`
- `chief-of-staff/feedback-loop.skill.md`

## Promotion Rule

Do not add a skill to an agent by default until it improves at least one
scenario metric without weakening permissions, honesty, or artifact validation.
