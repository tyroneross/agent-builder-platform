# Chief of Staff Feedback Loop Skill

## Purpose

Improve weekly planning by comparing planned work against actual output.

## Inputs

- Weekly plan.
- Completed artifacts.
- Calendar actuals.
- Open-loop log.
- User feedback.

## Outputs

- Planned-vs-actual review.
- Learning ledger update.
- Promoted lessons.
- Rejected lessons.

## Procedure

1. Compare planned deep-work blocks with actual completed work.
2. Track which handoffs reduced user effort and which created rework.
3. Identify one schedule pattern to keep, one to revise, and one to remove.
4. Promote lessons only when they improve measured output without new trust,
   health, or security costs.
5. Carry forward unresolved open loops with owners and dates.
