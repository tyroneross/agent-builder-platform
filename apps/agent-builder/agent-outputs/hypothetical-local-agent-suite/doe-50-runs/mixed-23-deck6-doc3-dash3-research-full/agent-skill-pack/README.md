# Agent Skill Pack

- chief-of-staff.100x-productivity-planning - strength-focused weekly planning

- shared.local-model-routing - model selection by latency and quality
- shared.local-llm-doe - small-model experiment design, replication, and cautious interpretation

- shared.document-structure - Word document sections, tables, and appendices
